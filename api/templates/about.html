使用的科技

使用了很多现代化的科技，以下是一些被使用的科技

网站代码

Bootstrap

Bootstrap 是个开源的CSS和JS框架，减少了我们开发网站的时间

Leaflet

Leaflet 是个开源的JS框架，让我们在网站中加入了地图

Flask

Flask 是个开源的Python框架，让我们把我们的代码变得更整洁，运行时更快，还让我们自定我们网站的链接

Vercel

Vercel 是我们网站的服务器提供者，它建立在亚马孙云科技的框架上，让我们把服务器带到新加坡，给公众提供一个更快的网站
